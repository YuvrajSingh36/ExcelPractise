Project 5 Kennels & Carriers

An animal-supply inventory workbook. Practise pie charts, expanding chart data ranges, chart styles, icon sets and hyperlinks across worksheets — the classic MO-210 chart customisation tasks.

Task 1

ChartsPie

On the Inventory worksheet, create a 2-D Pie chart using the Type and Total columns (ranges B3:B15 and F3:F15). Move the chart to a new sheet named Inventory Chart.

Show solution steps
  1. Select B3:B15, hold Ctrl and select F3:F15.
  2. Insert → Charts group → Insert Pie or Doughnut Chart → 2-D Pie.
  3. Chart Design → Move Chart → New sheet → type Inventory Chart → OK.

Task 2

ChartsStyle

Apply Chart Style 4 and add the chart title Inventory by Type.

Show solution steps
  1. Click the chart → Chart Design tab → Chart Styles gallery → choose Style 4.
  2. Click the chart title placeholder and type Inventory by Type.

Task 3

ChartsData Range

A new row has been added for item 9001 Free-range Chicken Pen. Expand the chart's data range to include this new row so the slice appears in the pie.

Show solution steps
  1. Click the chart → Chart Design → Select Data.
  2. Edit the Chart data range so it ends at row 16 (e.g. =Inventory!$B$3:$B$16, =Inventory!$F$3:$F$16) → OK.
  3. Or drag the coloured range handles on the source sheet down by one row.

Task 4

CellsIcon Set

On the Inventory sheet, apply the 3 Flags conditional formatting icon set to the Inventory column (D4:D15) so low stock is highlighted.

Show solution steps
  1. Select D4:D15.
  2. Home → Conditional Formatting → Icon Sets → 3 Flags.

Task 5

WorkbooksHyperlink

On the Inventory sheet, on the cell containing the dagger note for Free-range Chicken Pen†, add a hyperlink to cell A1 on the Wire Fencing worksheet with screen tip Compatible fencing.

Show solution steps
  1. Right-click the cell → Link (or Ctrl+K).
  2. Place in This Document → choose Wire Fencing → cell reference A1.
  3. ScreenTip… → enter Compatible fencing → OK → OK.

Task 6

ChartsLayout

On the Inventory Chart sheet, add Data Labels → Outside End and remove the legend.

Show solution steps
  1. Select the chart → Chart Design → Add Chart Element → Data Labels → Outside End.
  2. Add Chart Element → Legend → None.
