Project 7 Specialty Beverages

A flavour-profit workbook. Practise workbook-wide Find & Replace, switching row/column on a chart, chart styles and colours, axis titles, gridlines and series formatting.

← All projects

Task 1

CellsFind/Replace

Across the entire workbook, use Find & Replace to change every instance of Choco to Chocolate, matching case.

Show solution steps
  1. Press Ctrl+H → Options → set Within: Workbook, tick Match case.
  2. Find what: Choco, Replace with: Chocolate → Replace All.

Task 2

Charts

Create a Clustered Column chart of Flavors against Total Profit.

Show solution steps
  1. Select the Flavors column, Ctrl+select Total Profit column.
  2. Insert → Insert Column or Bar Chart → Clustered Column.

Task 3

ChartsSwitch

On the chart, Switch Row/Column so flavors appear in the legend instead of the axis.

Show solution steps
  1. Click the chart → Chart Design → Switch Row/Column.

Task 4

ChartsStyle

Apply Chart Style 7 and change the chart colour to Monochromatic Palette 3.

Show solution steps
  1. Chart Design → Chart Styles → Style 7.
  2. Chart Design → Change Colors → Monochromatic Palette 3.

Task 5

ChartsElements

Add a Primary Vertical axis title Profit ($) and turn on Major & Minor Gridlines for the vertical axis.

Show solution steps
  1. Chart Design → Add Chart Element → Axis Titles → Primary Vertical → type Profit ($).
  2. Add Chart Element → Gridlines → Primary Major Vertical and Primary Minor Vertical.

Task 6

ChartsFormat

Format the data series so the columns use a solid fill in Orange, Accent 2, and move the chart to a new sheet named Profit Chart.

Show solution steps
  1. Right-click any column → Format Data Series → Fill → Solid fill → Color: Orange, Accent 2.
  2. Chart Design → Move Chart → New sheet → Profit Chart → OK.
