Project 6 Utah National Parks

Q2 visitor attendance for the Mighty 5. Practise table styles, total rows, pie charts with quick layouts, alt text, and page-layout printing options.

← All projects

Task 1

Tables

On the Q2 Attendance worksheet, convert the range A3:E8 to a table with headers, then apply Orange, Table Style Medium 18.

Show solution steps
  1. Select A3:E8 → Insert → Table → tick My table has headers → OK.
  2. Table Design → Table Styles → choose Orange, Table Style Medium 18.

Task 2

TablesTotal Row

Add a Total Row to the table and show the SUM in the Total column.

Show solution steps
  1. Click in the table → Table Design → tick Total Row.
  2. In the Total column's total cell, choose Sum from the dropdown.

Task 3

ChartsPie

Create a 2-D Pie chart of each park's Total Q2 attendance (Park names vs Total). Place it as an object on the same worksheet.

Show solution steps
  1. Select Park names (A4:A8), Ctrl+select Total values (E4:E8).
  2. Insert → Insert Pie or Doughnut → 2-D Pie.

Task 4

ChartsLayout

Apply Quick Layout 2 and change the chart title to Q2 Visitor Attendance.

Show solution steps
  1. Select the chart → Chart Design → Quick Layout → Layout 2.
  2. Click the chart title and type Q2 Visitor Attendance.

Task 5

ChartsAlt Text

Add alt text to the chart: Title Park Attendance, Description Pie chart comparing Q2 visitors by park.

Show solution steps
  1. Right-click the chart → View Alt Text…
  2. Type the title and description in the Alt Text pane.

Task 6

WorkbooksPage Layout

Set page orientation to Landscape, fit the sheet to 1 page wide by 1 tall, and add a footer with the file name in the centre section.

Show solution steps
  1. Page Layout → Orientation → Landscape.
  2. Page Layout → Scale to Fit → Width = 1 page, Height = 1 page.
  3. Insert → Header & Footer → click centre footer → File Name.
