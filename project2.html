Task 1

IFAutoFill

In cell F2, use the IF function to display "Bonus" if Sales (column E) is greater than 10000, otherwise "-". Fill down to F50.

Show solution steps
  1. In F2 enter =IF(E2>10000,"Bonus","-").
  2. Double-click the fill handle to copy down to F50.

Task 2

Conditional formatting

Apply conditional formatting to E2:E50 so cells greater than $15,000 are highlighted with green fill.

Show solution steps
  1. Select E2:E50.
  2. Home → Conditional Formatting → Highlight Cells Rules → Greater Than.
  3. Enter 15000 and choose Green Fill.

Task 3

Charts

Insert a Clustered Column chart based on A1:B12. Move it to a new sheet named RegionChart.

Show solution steps
  1. Select A1:B12.
  2. Insert → Charts → Clustered Column.
  3. Right-click chart → Move Chart → New sheet → name RegionChart.

Task 4

View

Freeze the top row so it stays visible while scrolling.

Show solution steps
  1. View → Freeze Panes → Freeze Top Row.

Task 5

Headers/Footers

Add the file path to the left footer of the Sales worksheet.

Show solution steps
  1. Insert → Header & Footer, click left footer area.
  2. Click File Path on the Header & Footer Elements group.
