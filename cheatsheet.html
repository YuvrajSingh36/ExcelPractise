Core functions

FunctionExampleUse
SUM=SUM(A1:A10)Total a range or named range
AVERAGE=AVERAGE(B2:B20)Mean
MIN / MAX=MAX(C:C)Smallest / largest
COUNT / COUNTA / COUNTBLANK=COUNTA(D2:D50)Count numbers / non-empty / empty
IF=IF(A1>0,"Yes","No")Conditional
SORT=SORT(A2:A20)Dynamic sort
UNIQUE=UNIQUE(B2:B100)Distinct values
CONCAT / TEXTJOIN=TEXTJOIN(", ",TRUE,A1:A5)Join text
UPPER / LOWER / LEN=UPPER(A1)Text formatting

Key shortcuts

  • Ctrl+T – Create table
  • Ctrl+Shift+L – Toggle filters
  • Alt+= – AutoSum
  • F4 – Toggle absolute references
  • Ctrl+1 – Format Cells dialog
  • Ctrl+; – Insert today's date
  • F12 – Save As

Common ribbon paths

  • Import XML: Data → Get Data → From File → From XML
  • Named range: Formulas → Define Name (or use Name Box)
  • Compatibility check: File → Info → Check for Issues
  • Page orientation: Page Layout → Orientation
  • Print titles: Page Layout → Print Titles
