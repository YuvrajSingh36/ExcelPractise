Task 1

Tables

Convert the range A1:E40 to a table with the style Blue, Table Style Medium 2. Use the table headers.

Show solution steps
  1. Select A1:E40, press Ctrl+T, tick My table has headers.
  2. On Table Design, choose Blue, Table Style Medium 2.

Task 2

Tables

Add a Total Row that shows the SUM of column Quantity.

Show solution steps
  1. Click in the table → Table Design → tick Total Row.
  2. In the Quantity column total cell, choose Sum.

Task 3

SORT

In G2, use the SORT function to return the table column Product in ascending order.

Show solution steps
  1. =SORT(Table1[Product])

Task 4

UNIQUE

In H2, use UNIQUE to list each distinct Category.

Show solution steps
  1. =UNIQUE(Table1[Category])

Task 5

Print

Set rows 1 to repeat at the top when printed.

Show solution steps
  1. Page Layout → Print Titles → Rows to repeat at top: $1:$1.

Task 6

Tables

Convert the table back to a normal range.

Show solution steps
  1. Click in the table → Table Design → Convert to Range → Yes.
