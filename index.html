Exam objective domains

1. Worksheets & Workbooks

Import data (CSV, XML, web), navigate via named ranges, modify page setup, headers/footers, print area, orientation, inspect for issues.

2. Cells & Ranges

Paste options, AutoFill, fill series, formats, conditional formatting, define & reference named ranges.

3. Tables

Create tables from ranges, apply table styles, banded rows, total row, sort/filter, convert to range.

4. Formulas & Functions

SUM, AVERAGE, MIN, MAX, COUNT/COUNTA/COUNTBLANK, IF, SORT, UNIQUE, CONCAT, TEXTJOIN, LEN, UPPER, LOWER.

5. Charts

Create charts, switch row/column, add elements, apply chart styles & layouts, alt text.
