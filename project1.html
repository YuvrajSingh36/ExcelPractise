Task 1

ImportXML

Import the data from the source file MoreRentals.xml located in the GMetrixTemplates folder. Load the Diving_Rentals data to a new worksheet. (Accept all other defaults).

Show solution steps
  1. Go to Data → Get Data → From File → From XML.
  2. Navigate to the GMetrixTemplates folder and select MoreRentals.xml.
  3. In the Navigator, choose Diving_Rentals.
  4. Click Load → Load To… and select New worksheet, then OK.

Task 2

Cells & Ranges

On the Surf_Rentals worksheet, clear the formatting in cell range A4:D4.

Show solution steps
  1. Select A4:D4.
  2. On the Home tab → Editing → Clear → Clear Formats.

Task 3

FormulasNamed ranges

On the Surf_Rentals worksheet in cell C15, enter the rental cost of a Full Day Boogie Board Package using the SUM function and the named range Boogie_Package.

Show solution steps
  1. Click cell C15.
  2. Type =SUM(Boogie_Package) and press Enter.
  3. Tip: To verify the named range, use Formulas → Name Manager.

Task 4

Page layout

Change the orientation of the Surf_Rentals worksheet to Landscape.

Show solution steps
  1. Open the Surf_Rentals sheet.
  2. Go to Page Layout → Orientation → Landscape.

Task 5

Inspect

Inspect the workbook for compatibility issues. Copy the results to a new worksheet.

Show solution steps
  1. Click File → Info → Check for Issues → Check Compatibility.
  2. In the dialog, click Copy to New Sheet, then OK.
  3. A new Compatibility Report sheet will be created.
